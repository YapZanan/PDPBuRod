Hal yang perlu didownload:
1. CMAKE (https://cmake.org/download/)
2. Visual Studio (https://visualstudio.microsoft.com/downloads/)
3. Visual Studio Code (https://code.visualstudio.com/download)
4. Git (https://git-scm.com/downloads)


Langkah instalasi:
1. Install satu persatu instalasi yang sudah didownload,
2. Setelah semua selesai terinstall, buka Visual Studio, lalu centang module desktop development with c++,
3. Buka VSCode, tekan ctrl+shift+p untuk membuka command pallete, atau bisa juga ke view>Command Pallete,
4. Ketikkan Clone, akan muncul pilihan untuk melakukan clone git
5. Copy Link yang ada di project ini (https://github.com/YapZanan/PDPBuRod.git), lalu tekan enter,
6. Tekan Ctrl+Shit+` untuk membuka terminal di Visual Studio Code, atau bisa dengan menekan Terminal>New Terminal,
7. Pada terminal, ketik "pip install -r requirements.txt", lalu tekan enter,
8. Semua modul yang dibutuhkan akan diinstall, tunggu sampai selesai.


Untuk menambah subjek, buka folder belajar dan tambahkan foto di situ, nama pada foto tersebut akan menjadi nama yang tercetak dalam presensi
