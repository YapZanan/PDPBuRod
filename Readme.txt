1. Pycharm (https://www.jetbrains.com/pycharm/)
2. CMAKE (https://cmake.org/download/)
3. Visual Studio CPP (https://visualstudio.microsoft.com/visual-cpp-build-tools/)

Untuk langkah- langkah di atas, tutorialnya bisa dilihat di sini
https://medium.com/analytics-vidhya/how-to-install-dlib-library-for-python-in-windows-10-57348ba1117f

Setelah itu, buka Pycharm dan buat project baru

Buka File>Setting>Interpreter, lalu pada tombol "+", install beberapa package yang ada di bawah
1. face-recognition
2. PyQt5
3. PyQt5-Qt
4. PyQt5-Qt5
5. numpy
6. csv
7. cv2

Setelah semua package berhasil di install, copy semua file di folder yang saya berikan ke file project python yang baru dibuat
(klik kanan pada nama project di PyCharm>open in>explorer)

Setelah itu, buka pycharm dan jalankan file mainwindow.py (klik 2 kali pada bagian project, lalu tekan tombol start di kanan atas)

Untuk menambah subjek, buka folder belajar dan tambahkan foto di situ, nama pada foto tersebut akan menjadi nama yang tercetak dalam presensi
